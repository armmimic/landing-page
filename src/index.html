Privacy

You’re in full control of who is accessing your data.

Privacy

Gaining full control over their data on Lumeos, the users can decide whom to grant the access to the information. This is easily done with the Lumeos intuitive user interface. The user can provide the key to access certain information to other parties of the user’s own choice, and no one else will be able to use the data, even the platform. Lumeos records all of the data to the user’s personal account, leaving no traces of it for the platform to use.

Ownership

You own all the data generated on the platform.

Ownership

On mainstream networks, data is owned by the network. The platform is supposed to only use data under the user’s consent, but the latest media coverings show it is not fairly so. The Lumeos platform doesn’t own any user data. The user providing the network with the data remains the data owner and the only one managing the access to this data, and this point is secured by the incorruptible blockchain system. Provided with the key to the Lumeos account, the user gains full control over the data and is free to store, use, share or sell it.

Data monetization

Earn tokens by allowing users to view some of your data.

Data monetization

Now that the user owns their data, they can sell it to anyone that requests it. They have the option to anonymize it as well, so that it’s only available as part of an aggregated data. (for instance a user can buy the result of a 100-answered poll but they cannot see what each individual user has answered)

Proof of Friendship

A social network can not be complete without friends. Lumeos introduces the concept of Proof of Friendship that incentivizes users to get their friends to use the platform through referrals. Friends can compete with each others to establish streaks of login to the app, as well as see who has more friends, more tokens, and best created polls.
